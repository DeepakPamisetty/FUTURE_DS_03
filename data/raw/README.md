# Raw Data

Place the Kaggle competition file here after downloading it from:

<https://www.kaggle.com/competitions/ga-customer-revenue-prediction/data>

Supported filenames:

- `train_v2.csv`
- `train.csv`

The analysis pipeline also supports:

- `data/raw/ga-customer-revenue-prediction/train_v2.csv`
- `data/raw/ga-customer-revenue-prediction/train.csv`

Kaggle competition files are not committed to this repository.
